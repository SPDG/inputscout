# InputScout brand assets

The InputScout mark combines a rounded keyboard key with a radar sweep. The
key represents input hardware; the sweep and target represent discovery,
telemetry, and device awareness.

## Assets

- `logo-mark.svg` — square application and avatar mark
- `logo-lockup.svg` — horizontal lockup for light backgrounds
- `logo-lockup-dark.svg` — horizontal lockup for dark backgrounds
- `logo-mark-512.png` — raster application icon
- `logo-lockup.png` — raster preview of the light lockup

The SVG files are the source assets. Raster exports should always be generated
from them.

## Palette

| Role | Color | Hex |
| --- | --- | --- |
| Midnight | Deep background | `#0B1220` |
| Scout teal | Sweep and active state | `#2DD4BF` |
| Signal blue | Discovery arcs | `#38BDF8` |
| Cloud white | Highlight | `#F8FAFC` |
| Slate | Supporting copy | `#64748B` |

## Usage

Keep clear space around the mark equal to at least one quarter of its width.
Do not recolor individual elements, stretch the artwork, add effects, or place
the mark inside another container shape.

InputScout is an independent project and is not affiliated with or endorsed by
any supported device manufacturer.
